    1. `resolveCrmLink(contactWaId)` — auto-link logic (FOUND-05):
       - if `!hasKv()` return `{ crmClientId: null, crmLeadId: null }`
       - `const clientId = await kvGet(\`clientByPhone:\${contactWaId}\`)` (contactWaId already normalised by caller)
       - if no clientId: return `{ crmClientId: null, crmLeadId: null }`
       - else: `const recent = await kvZRevRange(\`client:\${clientId}:leads\`, 0, 0)` ; `const crmLeadId = recent[0] || null`
       - return `{ crmClientId: String(clientId), crmLeadId }`

    2. `upsertThread({ contactWaId, teamWaId, contactName, lastMessageAt, preview, crmClientId, crmLeadId })` — returns the thread id (uuid):
       ```js
       const sql = db();
       const rows = await sql`
         INSERT INTO whatsapp_threads
           (contact_wa_id, team_wa_id, contact_name, crm_client_id, crm_lead_id, last_message_at, last_message_preview, unread_count)
         VALUES (${contactWaId}, ${teamWaId}, ${contactName}, ${crmClientId}, ${crmLeadId}, ${lastMessageAt}, ${preview}, 1)
         ON CONFLICT (contact_wa_id, team_wa_id) DO UPDATE SET
           last_message_at = EXCLUDED.last_message_at,
           last_message_preview = EXCLUDED.last_message_preview,
           unread_count = whatsapp_threads.unread_count + 1,
           contact_name = COALESCE(EXCLUDED.contact_name, whatsapp_threads.contact_name),
           crm_client_id = COALESCE(whatsapp_threads.crm_client_id, EXCLUDED.crm_client_id),
           crm_lead_id = COALESCE(whatsapp_threads.crm_lead_id, EXCLUDED.crm_lead_id)
         RETURNING id
       `;
       return rows[0].id;
       ```

    3. `insertInboundMessage({ wamid, threadId, fromWaId, toWaId, messageType, body, timestampMs })`:
       ```js
       const sql = db();
       await sql`
         INSERT INTO whatsapp_messages
           (wamid, thread_id, direction, from_wa_id, to_wa_id, message_type, body, timestamp_ms)
         VALUES (${wamid}, ${threadId}, 'inbound', ${fromWaId}, ${toWaId}, ${messageType}, ${body}, ${timestampMs})
         ON CONFLICT (wamid) DO NOTHING
       `;
       ```

       - `await insertInboundMessage({ wamid, threadId, fromWaId: contactWaId, toWaId: teamWaId, messageType: type, body: body || null, timestampMs: Number(timestampMs) });`
       - return `{ threadId, crmLeadId, crmClientId }`

    5. `linkThreadToLead(threadId, { crmClientId, crmLeadId })` — for later re-linking (used in tests + future phases): UPDATE whatsapp_threads SET crm_client_id, crm_lead_id WHERE id = threadId.
